NADA-CPT - A New Objective Diagnostic Tool for ADHD

Developed by Mehmet Sevri, Ahmet Özaslan, and collaborators

Project Overview

This project is supported by TÜBİTAK under the 1005 program, Project Code: 119S187. The NADA-CPT (Distractor-embedded Auditory Continuous Performance Test) is a novel web-based diagnostic tool specifically designed to assess core clinical symptoms of Attention-Deficit/Hyperactivity Disorder (ADHD) such as inattention, response inhibition, hyperactivity, and timing. By integrating auditory and visual distractors, the tool offers a highly ecologically valid approach for diagnosing ADHD in children aged 6-12 years.

About the Application

The NADA-CPT application provides an auditory-based test environment with embedded distractors, simulating real-world attention-demanding scenarios, such as those found in a classroom. This approach enhances the accuracy of ADHD diagnostics and allows for a more comprehensive understanding of attentional challenges in a clinical setting. The use of both visual and auditory distractors helps assess selective attention, impulsivity, and response inhibition, making NADA-CPT a valuable tool in ADHD evaluation.
